🚀 Quick Setup

1. Install Dependencies

npm install

2. Environment Setup

cp .env.example .env

Configure SMTP settings for email authentication:

SMTP_HOST=smtp.gmail.com
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password

3. Database Setup

npm run db:generate
npm run db:push
npm run db:seed

4. Start Development

npm run dev
npm run workers:all

📧 Email Authentication System

How It Works

  • User enters email address
  • System sends 6-digit OTP code
  • User verifies OTP to authenticate
  • New users are automatically registered
  • JWT token issued for session management
