🚀 Quick Setup

1. Install Dependencies

npm install

2. Environment Setup

cp .env.example .env

Configure database and services:

DATABASE_URL="postgresql://user:pass@localhost:5432/carbooking"
REDIS_URL="redis://localhost:6379"
JWT_SECRET="your-secure-jwt-secret"

3. Database Setup

npm run db:generate
npm run db:push
npm run db:seed

4. Start Development

npm run dev
npm run workers:all
